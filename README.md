# Match-Analysis-Barcelona-vs-Newcastle-Shot-xG-Breakdown
# ⚽ Barcelona vs Newcastle — Shot & xG Analysis

This project analyzes shot data from a recent UEFA Champions League match, focusing on chance creation, shot quality (xG), and team attacking patterns.

---

## 📊 Project Overview

The goal of this project is to use event-level football data to:
- Analyze how both teams created scoring opportunities
- Evaluate shot quality using Expected Goals (xG)
- Identify key players and tactical patterns
- Visualize insights using Python

---

## 🔍 Key Insights

- The home team dominated attacking play, generating more shots and higher-quality chances.
- Most shots were taken from central areas inside the penalty box, indicating effective build-up play.
- The away team created fewer opportunities and relied on more sporadic attacking moments.
- Key players such as Lamine Yamal and Robert Lewandowski generated the highest xG for the home side.
- Anthony Elanga was the primary attacking threat for the away team.

---

## 📈 Visualizations

### 1. Shot Maps
- Separate shot maps for home and away teams
- Circle size represents xG (chance quality)

### 2. xG per Player
- Highlights players generating the most dangerous chances

### 3. Total Shots per Player
- Shows shot volume and attacking involvement

### 4. Shot Situations
- Breakdown of how chances were created (assisted, fast-break, set-piece)

### 5. Shot Outcomes
- Distribution of goals, saves, misses, and blocks

## 🛠️ Tools & Technologies

## 📁 Dataset

- Event-level shot data extracted from SofaScore
- Includes:
  - Player
  - Shot location (x, y)
  - xG (Expected Goals)
  - Shot type (goal, save, miss, block)
  - Situation (assisted, fast-break, etc.)
